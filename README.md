#Hero Stuffing Agency

Even Heroes need a work!

## Pre-requirements
###Front-end:
* node ^4.x
* npm ^3.x
* bower ^1.6.x

###Back-end:
* dnvn ^1.0.0-rc2-15546
* dnu
* dnx

## Building project

In project folder run:
###For front-end:
* npm i
* bower i
* npm run build

###Back-end:
* dnu restore
* dnx web

Have fun!
